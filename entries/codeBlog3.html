Code Blog 3
published on (11/09/2016) by Teresita C. G. Nader

  • What was challenging about the assignment, and what specific kinds of problems did you have. How did you solve those problems?

    • It was difficult to create the web page without npms, because all the tutorials I found online used npms, so it was difficult to do the web page without npms because I didn't knew how to do it without npms. Most tutorials and the web page provided for the assignment wasn't clear enough for a begginer like me.

  • What did you learn from the assignment? Were there any special insights you had? What did you find that you already knew?

    • I learned that npms are very useful tools to build web sites, they will make shorter the time you spend creating the components of a web page.

  • What kinds of resources were helpful for completing the assignment? Specific websites? Lectures? The class Piazza forum? The TAs? How did you use each of these resources?

    • The YouTube videos for Node.js Beginners from thenewboston were very helpful to start the assignment, and understand how node.js works. To finish the assignment it was very helpful to work with my classmates.

  • What are one or two things you had to Google to complete the assignment?

    • How to use Node.js, how to do a web page without npms and how to make if/if else statements on JavaScript.
